# apafis-edit
 Logiciel permettant de retirer ou modifier le versioning d'un fichier généré par apafis

Ce dépôt vise à fournir une solution aux personnes travaillant avec l'application APAFIS, pour modifier le numéro de version ou d'enregistrement.
Il peut ainsi fournir aux responsables des Comités d'Ethique ou de l'enregistrement des saisines dans l'interface Apafis une solution permettant de faciliter leur travail.

L'exécutable et le script perl fournis dans ce dépôt permettent de modifier ou de supprimer les informations de versioning enregistrées lors de l'utilisation de la fonction Sauvegarder définitivement.

### Mode d'emploi :

# Version basique
Télécharger le script exe (windows) ou .pl (linux ou MacOs)
* Placer la saisine générée par APAFIS dans le répertoire contenant l'exécutable ou le script perl.
* Lancer le logiciel saisine-versioning v1.1.exe ou executer 'perl saisine-versioning v1.1.pl'
* Suivre les instructions à l'écran

# Version graphique
Télécharger le script exe (windows) ou .pl (linux ou MacOs)
* Lancer le logiciel Apafis-Visual-Editor-v1.00.exe ou executer 'perl Apafis-Visual-Editor-v1.00.pl'
* Utiliser l'interface
